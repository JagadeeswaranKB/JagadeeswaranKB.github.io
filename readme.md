# Portfolio Website

This repository contains the source code for my personal portfolio website built using HTML, CSS, and JavaScript.

## Overview

This portfolio website serves as a platform to showcase my skills, projects, achievements, and contact information. It consists of the following sections:

- **Home**: Introduction section with a brief overview and links to resume and social media profiles.
- **About**: Information about myself, including my interests, skills, and goals.
- **Skills**: A list of my professional skills with graphical representations.
- **Work**: Display of my past projects with clickable images for more details.
- **Contact**: A contact form for visitors to reach out to me.

## Features

- Responsive design to ensure compatibility across various devices and screen sizes.
- Smooth scrolling navigation for seamless user experience.
- Contact form powered by FormSubmit for easy communication.
- Integration of social media profiles for networking opportunities.

## Technologies Used

- HTML5
- CSS3
- JavaScript
- jQuery
- Font Awesome Icons
- ScrollReveal.js

## Setup

To run this project locally, simply clone the repository and open the `index.html` file in a web browser.

```bash
git clone https://github.com/JagadeeswaranKB/JagadeeswaranKB.github.io.git
```
